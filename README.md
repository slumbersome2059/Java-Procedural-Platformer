- To run the game run the DesktopLauncher(this is inside /desktop/src/com.mygdx.game/DesktopLauncher) once you open the file in an IDE
- Press W for up, A for left, S for down, D for right
- In the level with the different towers, first click on the screen to activate torch then use the mouse to move the torch
- The player only experiences gravity in the level with the towers, everywhere else there is no gravity# Java-Procedural-Platformer
